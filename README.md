# 🔍 Fake Account Detection Using Machine Learning and Data Science

## 📌 Overview

This project focuses on detecting fake accounts on social media platforms using Machine Learning and Data Science techniques. With the rapid growth of online platforms, fake accounts are widely used for spam, misinformation, and malicious activities.

The system leverages advanced ML algorithms to analyze user behavior and classify accounts as **genuine or fake** with high accuracy.

---

## 🚀 Features

* 📊 Detects fake social media accounts using ML models
* 🧠 Uses **Artificial Neural Network (ANN)** and Gradient Boosting concepts
* 🔍 Analyzes key features:

  * Engagement Rate
  * Spam Commenting Behavior
  * Artificial Activity
* 📈 Displays **Accuracy & Loss Graphs**
* 🖥️ GUI-based system using Tkinter
* ⚡ Real-time prediction on new datasets

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** TensorFlow / Keras, Scikit-learn, Pandas, NumPy, Matplotlib
* **GUI:** Tkinter

---

## ⚙️ Project Workflow

1. Upload Social Network Dataset
2. Preprocess Dataset (cleaning & splitting)
3. Train ANN Model
4. Evaluate Accuracy & Loss
5. Predict Fake/Genuine Accounts
